Photonvision Libraries
===========================

robotpy's photonvision library has been deprecated and replaced with 
the vendor's supported photonlibpy package.

photonlibpy may be install via PIP: `pip install photonlibpy`

Refer to the `Photonvision site <https://docs.photonvision.org/>`_ for information
on using the API.
